> 本文件用于沉淀讨论后逐步收敛的方向、决策和待办。与 README 的"历史材料"不同，这里记录**当前最新结论**和**下一步要做的事**。
> 最近更新：2026-06-14

---

## 0. 一句话定位

> 让 LLM 把一个抽象概念，改写成**不点破术语、但内部机制忠实**的隐性叙事类比，并显式给出"情节↔机制零件"的对齐。

学术对象不是"寓言"，而是 **narrative analogy with a concealed target concept**。

---

## 1. 已拍板的方向（核心决策）

### 1.1 走窄版 M2NA，不走宽版 Concept-to-Fable

采用 **M2NA（Mechanism-to-Narrative Analogy Generation）** 的形式化定义：

```text
输入：
  c    目标概念
  G_c  概念的机制图 (V_c, E_c)：关键零件 + 它们的因果/强化/转化等关系
  T_c  禁用词集合：概念名 + 高泄露术语

输出：
  N    隐性叙事类比（短、具体、不出现 T_c）
  A    结构对齐：把 G_c 的节点/边映射到 N 的实体/事件/关系
```

四个成功条件（缺一不可）：

1. **Mechanism Preservation** 机制保持：把关键零件和关系都演出来。
2. **Lexical Concealment** 词汇隐藏：正文不出现概念名和术语。
3. **Analogical Alignability** 可对齐：每段情节能指回某个机制零件。
4. **Narrative Minimality** 叙事最小：短、具体、不堆砌文学性。

### 1.2 去寓言化

"寓言"背着三样不可评测的东西：文学性、道德教训、童话趣味。**全部砍掉**，改称 narrative analogy / explanatory scenario。只保留可自动化的硬指标：

- 机制覆盖率（Concept Coverage）
- 术语泄露率（Hard / Soft Leakage Rate）
- 映射一致性（Mapping Consistency）

### 1.3 定位 = 窄定义 + 保留教学动机 + 最小真人实验兜底

- 形式化定义和自动评估作为**主贡献**。
- "帮人学习"作为 **motivation**，不作为主证明。
- 只做**小规模**真人理解/偏好实验，用来给 concealment 这个约束提供合法性；**不做**大规模 retention（24h / 1 周）study。

## 2. 真正的护城河：concealment 三件套

调研结论：学界**没有人做生成侧的"刻意隐藏概念术语"**。
（ARN / NARB 等是去*识别*隐藏类比，不是生成时刻意隐藏。）

> **"机制图 G_c + 词汇隐藏 T_c + 显式对齐 A" 这个组合是空白的，是本文唯一的硬区分点。**

两层含义：

1. **选哪个领域是次要问题**，守住 concealment + 结构对齐才是命。
2. concealment 没人做 = 没有现成动机背书。必须用认知论证立住：
   > 显式术语会让学习者一看到"哦这是 X"就跳过推理；隐性叙事强迫 inference，因此更利于理解和迁移。
   这个论证正是最小真人实验要验证的东西。

## 3. 竞品 / 撞车地图（新颖性侦察结论）

| 方向 | 占位 | 代表工作 |
|---|---|---|
| 理科(物理/生物/化学)教育类比 | 🔴 极度饱和 | SCUA(EMNLP'24)、Unlocking Scientific Concepts(CHI'25)、Can LLMs Teach Human Learners(PMLR'25) |
| CS/ML 概念类比 | 🟠 已被碰且高度重合 | TU Delft 的 Gradient Descent 类比+真人A/B+Concept Coverage/Mapping 指标；ANVIL(CS类比+视频, 2026)；recursion 类比 |
| 通用概念解释 benchmark | 🟠 已有大规模 | LLM-as-an-Explainer(ACL ARR'26)：31,160 条解释，6 学科含 CS/Finance/社科 |
| 教育类比方法 pipeline | 🟠 强竞品 | Teaching Through Analogies(2026)：SMT 四阶段 source→sub-concept→explanation→eval（≈KG-Fable 流水线） |
| 科学故事化 + 评估 | 🟠 | StoryScore(2026)、Narrativity in AI Explanations(2026, 叙事四属性+7 自动指标) |
| 经济/社会科学概念类比生成 | 🟢 明显空白 | 仅有"LLM 做经济推理/计量"，无"生成类比教经济概念" |

**两个必须直接对线、写进 Related Work 划清界限的竞品**（已逐篇精读 → 见 [`论文调研/竞品精读_第一档.md`](论文调研/竞品精读_第一档.md)）：

- **TU Delft Gradient Descent 论文（2025）** —— 任务几乎撞（ML概念→类比→Coverage/Mapping指标→真人A/B）。但实为**本科毕业论文、样本15人、null result、专家一致性为负**，威胁等级低。我方区分：它单源域手工造类比、6概念、无 benchmark、**无 concealment、无机制图输入、无显式对齐输出**。它把"AI 能否生成比人工更好的类比"明确留作 future work = 我方切入口。
- **Teaching Through Analogies (2026)** —— 方法 pipeline 撞（SMT 四阶段，跨12模型）。我方区分：它做显式解释、不藏术语、不输入机制图、不要求 alignment 输出、无真人实验。可复用：SCAR/ParallelPARC 数据、LLM-judge"排序优于绝对分"经验、"结构化中间表示提升质量"作 G_c 旁证。

## 4. 领域选择（待最终拍板）

权衡：

- **AI/ML 概念**：Lin 本人是专家(G_c 质量可控)、被试好招，但**新颖性挤、动机弱**（AAAI 受众都懂 overfitting，藏术语的必要性低）。
- **经济/行为经济 + 社科机制概念**（path dependence、sunk cost、principal-agent、tragedy of commons…）：**最空白**，概念抽象反直觉、**类比收益最大、动机最易立**，但 Lin 非领域专家，G_c 和 faithfulness 判定需借权威知识/外部专家。

**当前倾向（待确认）：**

> 以**经济/社科机制概念为主战场** + **少量 AI/ML 子集**（用 Lin 专长保证一部分高质量样本），护城河锁死在 concealment + 机制图 + 对齐。

## 5. 下一个命门：G_c（机制图）从哪来

这是方法和评估能否立住的地基，**必须先解决**：

- **核心风险 = 循环评估**：若用 LLM 抽 G_c，再用 G_c 的节点算 Concept Coverage 去评 LLM 生成的覆盖率，就是自我指涉，审稿人必打。
- 待决策：G_c 是 **given（人工 gold，小而精，作为 benchmark 一部分）** 还是 **to-be-generated（模型能力的一部分）**。
- 当前倾向：**given + 人工校验**，从源头切断循环依赖，把"机制图正确性"从模型能力里剥离出去。
