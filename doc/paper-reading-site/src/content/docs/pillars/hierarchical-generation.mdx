# 层次化生成：先规划，再写作

通过 story planning、controllable generation 和 KG-guided generation，降低故事跑偏风险。

这一支柱负责把研究构想中的关键风险前置处理：如果没有可靠的生成流程，模型生成的寓言就容易变成流畅但不可控的故事。

## 在 Concept-to-Fable 中的作用

把生成过程拆成可控步骤，避免 LLM 直接写作时跑偏。
