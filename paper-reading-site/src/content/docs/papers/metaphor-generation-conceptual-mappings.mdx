# Metaphor Generation with Conceptual Mappings

**年份/来源**：2021 · ACL-IJCNLP  
**所属支柱**：[结构映射](/pillars/structure-mapping/)  
**PDF**：[打开本地 PDF](/papers/Method-02-MetaphorConceptualMappings-ACL-2021.pdf)  
**原文链接**：[https://aclanthology.org/2021.acl-long.524/](https://aclanthology.org/2021.acl-long.524/)

## 这篇论文在讲什么

研究基于概念映射生成隐喻表达。

## 与 Concept-to-Fable 的关系

它支撑“知识概念到寓言世界”的跨域映射思想。

## 推荐阅读部分

重点看 source domain 与 target domain 的映射如何表达。

## 可借鉴点

可借鉴概念映射，把科学实体映射为寓言角色，把关系映射为互动。

## 局限与注意事项

隐喻通常较短，不等同于有完整情节和教育目标的寓言。

## 研究导向 Q&A

### 1. 这篇论文解决的问题和 Concept-to-Fable 有什么关系？

它支撑“知识概念到寓言世界”的跨域映射思想。

### 2. 它支撑四大支柱中的哪一部分？

它主要支撑 **结构映射**。在 Concept-to-Fable 中，这一部分的任务是把抽象知识结构稳定映射成寓言中的角色、冲突和情节。

### 3. 我们可以借鉴它的什么方法、数据或评测方式？

可借鉴概念映射，把科学实体映射为寓言角色，把关系映射为互动。

### 4. 它没有解决什么，因此我们的任务还有什么研究空间？

隐喻通常较短，不等同于有完整情节和教育目标的寓言。 这也正是 Concept-to-Fable 可以继续推进的空间：把论文中的单点能力放进“课程 KG → 结构映射 → 寓言生成 → 教育评价”的完整链路里。

### 5. 如果把它用于“知识点转寓言”，需要怎样改造？

需要把短语级或句子级隐喻扩展为多事件寓言结构。
