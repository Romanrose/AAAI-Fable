# M2NA 2024-2026 相关论文检索

检索目标：围绕收窄后的任务 **Mechanism-to-Narrative Analogy Generation (M2NA)** 寻找 2024-2026 相关工作。

M2NA 当前定义：

```text
Input:
  target concept c
  mechanism graph G_c = (V_c, E_c)
  forbidden lexical set T_c

Output:
  implicit narrative analogy N
  alignment A between concept mechanism and narrative structure
```

核心约束：

- Mechanism Preservation
- Lexical Concealment
- Analogical Alignability
- Narrative Minimality

结论：2024-2026 已有很多相邻工作，但目前没有看到完全同构的任务。最接近的是 narrative analogy / analogy generation / educational analogy / graph-to-text 四条线。

## 1. 最接近 M2NA 的文献

### 1.1 ARN: Analogical Reasoning on Narratives

- 年份/venue：TACL 2024
- 链接：https://aclanthology.org/2024.tacl-1.59/
- 关键词：narrative analogy, surface mapping, system mapping, far/near analogy
- 核心内容：提出 Analogical Reasoning over Narratives (ARN)，用 narrative elements 建立 surface/system mappings，测试 LLM 是否能识别 narrative-level system analogies。
- 和 M2NA 的关系：非常近。它研究“叙事之间的类比识别”，你研究“概念机制到隐性叙事类比的生成”。
- 可借鉴：
  - surface mapping vs system mapping
  - far analogy / near analogy / distractor 设置
  - 人类与 LLM 的类比识别差距
- 差异：
  - ARN 是 narrative-to-narrative recognition。
  - M2NA 是 mechanism-to-narrative generation。
  - ARN 不要求词汇隐藏，也不输入抽象概念机制图。

### 1.2 ParallelPARC: A Scalable Pipeline for Generating Natural-Language Analogies

- 年份/venue：NAACL 2024
- 链接：https://aclanthology.org/2024.naacl-long.329/
- 关键词：paragraph-based analogy, scientific process analogy, ProPara-Logy
- 核心内容：用 LLM 创建复杂的段落级类比和 distractors，构建科学过程之间的类比数据集。
- 和 M2NA 的关系：非常近，特别是“科学过程”本身就是 mechanism-like structure。
- 可借鉴：
  - paragraph-based analogies
  - LLM data generation + human-validated gold set + silver set
  - challenging distractor design
- 差异：
  - 它主要做 analogy recognition 数据构建。
  - M2NA 做受控生成，并要求隐藏目标概念词和输出 alignment。

### 1.3 Boosting Scientific Concepts Understanding: Can Analogy from Teacher Models Empower Student Models?

- 年份/venue：EMNLP 2024
- 链接：https://aclanthology.org/2024.emnlp-main.346/
- 关键词：scientific concept understanding, teacher LM, student LM, analogy
- 核心内容：研究 teacher LM 生成的类比能否帮助 student LM 理解科学概念，并提升科学问答。
- 和 M2NA 的关系：非常适合支撑“类比可以帮助概念理解”的任务动机。
- 可借鉴：
  - scientific concept understanding 任务设置
  - analogy-as-learning-signal baseline
  - teacher-generated analogy baseline
- 差异：
  - 它关注模型理解，不是人类学习。
  - 它生成的是 free-form analogy，不是隐性叙事类比。

### 1.4 Unlocking Scientific Concepts: How Effective Are LLM-Generated Analogies for Student Understanding and Classroom Practice?

- 年份/venue：arXiv 2025，CHI 2025 conditional acceptance
- 链接：https://arxiv.org/abs/2502.16895
- 关键词：LLM-generated analogies, science education, classroom practice
- 核心内容：通过 high school students 和 teachers 的两阶段研究，评估 LLM 生成类比对 biology / physics 概念理解的作用，并开发教师生成/修改类比系统。
- 和 M2NA 的关系：非常适合支撑教育意义和 human evaluation 设计。
- 可借鉴：
  - controlled in-class test
  - teacher refinement
  - over-reliance / overconfidence 风险
- 差异：
  - 它是 HCI/education 系统研究。
  - M2NA 是 NLP 生成任务，主评估更关注机制保持、词汇隐藏和结构映射。

### 1.5 Past Meets Present: Creating Historical Analogy with Large Language Models

- 年份/venue：ACL 2025
- 链接：https://aclanthology.org/2025.acl-long.200/
- 关键词：historical analogy, analogy acquisition, self-reflection
- 核心内容：给定当代事件，检索或生成类似历史事件，并用 self-reflection 减少 hallucination 和 stereotype。
- 和 M2NA 的关系：相近在“acquiring / generating analogies for unfamiliar events”。
- 可借鉴：
  - retrieval vs generation baseline
  - self-reflection for hallucination/stereotype mitigation
  - automatic multi-dimensional assessment
- 差异：
  - 它是 event-to-event analogy。
  - M2NA 是 concept mechanism-to-narrative analogy。

### 1.6 Automatic Extraction of Metaphoric Analogies from Literary Texts

- 年份/venue：arXiv 2024，COLING 2025 long paper
- 链接：https://arxiv.org/abs/2412.15375
- 关键词：metaphoric analogy, literary text, implicit elements
- 核心内容：从文学文本中抽取 metaphoric analogies，评估 LLM 结构化隐喻映射和生成隐含元素的能力。
- 和 M2NA 的关系：很近，尤其是 implicit elements 和 structure metaphoric mappings。
- 可借鉴：
  - implicit analogy elements 的标注/评估
  - 从文本片段中结构化 mapping 的方法
- 差异：
  - 它是 extraction / interpretation。
  - M2NA 是 generation。

## 2. 类比推理与类比提示方向

### 2.1 Large Language Models as Analogical Reasoners

- 年份/venue：ICLR 2024
- 链接：https://proceedings.iclr.cc/paper_files/paper/2024/hash/4990dad2c1696224de42573d0222554a-Abstract-Conference.html
- 关键词：analogical prompting, self-generated exemplars
- 核心内容：提出 Analogical Prompting，让模型先自生成相关 exemplars / knowledge，再解决问题。
- 和 M2NA 的关系：可作为方法 baseline：让模型先生成类似案例，再写故事。
- 可借鉴 baseline：
  - Analogical Prompting
  - self-generated exemplar planning

### 2.2 Thought Propagation: An Analogical Approach to Complex Reasoning with LLMs

- 年份/venue：ICLR 2024
- 链接：https://proceedings.iclr.cc/paper_files/paper/2024/hash/2dae7d1ccf1edf76f8ce7c282bdf4730-Abstract-Conference.html
- 关键词：analogous problems, solution transfer, reasoning
- 核心内容：让模型提出并求解类似问题，再把类似问题的解法传播到原问题。
- 和 M2NA 的关系：可作为“类比规划”方法灵感。
- 可借鉴：
  - generate analogous situations first
  - reuse solution strategy / structure

### 2.3 ANALOGYKB: Unlocking Analogical Reasoning of LMs with A Million-scale Knowledge Base

- 年份/venue：ACL 2024
- 链接：https://aclanthology.org/2024.acl-long.68/
- 关键词：analogy KB, KG-derived analogies, analogy recognition and generation
- 核心内容：从知识图谱中构建百万级 analogy knowledge base，提升模型类比识别和生成能力。
- 和 M2NA 的关系：可作为 analogy source / retrieval baseline。
- 可借鉴：
  - KG-derived analogies
  - analogy recognition / generation evaluation
- 差异：
  - 不是叙事类比，不涉及隐性表达。

### 2.4 KnowledgePrompts: Proportional Analogies via Knowledge-Enhanced Prompting

- 年份/venue：COLING 2025
- 链接：https://aclanthology.org/2025.coling-main.268/
- 关键词：proportional analogy, knowledge-enhanced prompting
- 核心内容：构建 15K MCQA 类比补全数据集，评估 LLM 在 knowledge-enhanced prompt 下解决 proportional analogy 的能力。
- 和 M2NA 的关系：可支撑“知识增强对类比任务重要”的论点。

### 2.5 Self-supervised Analogical Learning using Language Models

- 年份/venue：arXiv 2025
- 链接：https://arxiv.org/abs/2502.00996
- 关键词：self-supervised analogical learning, transfer solution
- 核心内容：训练模型显式迁移高质量 symbolic solutions 到相似稀有案例，提高推理一致性和泛化。
- 和 M2NA 的关系：可作为未来 work 或训练型方法灵感，不是直接 baseline。

### 2.6 The Curious Case of Analogies: Investigating Analogical Reasoning in LLMs

- 年份/venue：AAAI 2026
- 链接：https://ojs.aaai.org/index.php/AAAI/article/view/40414
- 关键词：proportional analogy, story analogy, structural alignment, mechanistic interpretability
- 核心内容：用 proportional analogies 和 story analogies 分析 LLM 如何编码、应用和对齐关系信息。
- 和 M2NA 的关系：非常适合支撑“成功类比需要 structural alignment”。
- 可借鉴：
  - structural alignment 作为核心术语
  - success/failure 对齐分析
- 差异：
  - 它是理解/解释模型内部机制，不是生成任务。

### 2.7 Emergent Analogical Reasoning in Transformers

- 年份/venue：arXiv 2026，ICML 2026 spotlight
- 链接：https://arxiv.org/abs/2602.01992
- 关键词：analogical reasoning, Transformers, relational structure transfer, mechanistic analysis
- 核心内容：用 synthetic tasks 和机制分析研究 Transformer 如何实现类比推理，强调 relational structure transfer。
- 和 M2NA 的关系：理论支撑强，但不直接做文本生成。

## 3. 叙事生成 / 叙事解释方向

### 3.1 Are Large Language Models Capable of Generating Human-Level Narratives?

- 年份/venue：EMNLP 2024，Outstanding Paper
- 链接：https://aclanthology.org/2024.emnlp-main.978/
- 关键词：story arc, turning point, affective dimension, narrative evaluation
- 核心内容：评估 LLM 在 narrative development 和 plot progression 上的能力，发现 LLM 故事缺少 tension，结构同质化；加入 discourse features 可提升故事质量。
- 和 M2NA 的关系：可用于 narrative quality / diversity / template 问题的动机和评估。
- 可借鉴：
  - story arcs
  - turning points
  - affective dimensions
  - LLM story homogeneity

### 3.2 Can Stories Help LLMs Reason? Curating Information Space Through Narrative

- 年份/venue：Analogy-Angle 2025
- 链接：https://aclanthology.org/2025.analogyangle-1.8/
- 关键词：Story of Thought, narrative reasoning, metaphor, analogy
- 核心内容：提出 Story of Thought，让 LLM 用叙事方式组织问题信息，提升科学推理任务。
- 和 M2NA 的关系：支撑 narrative as reasoning/organization 的动机。
- 可借鉴：
  - narrative as specialized mode of thinking
  - metaphor/analogy in reasoning process
- 差异：
  - 它用故事帮助模型推理。
  - M2NA 生成故事帮助表达概念机制。

### 3.3 Tell me a story! Narrative-driven XAI with LLMs

- 年份/venue：Decision Support Systems 2025
- 链接：https://www.sciencedirect.com/science/article/pii/S016792362500003X
- 关键词：narrative-driven XAI, SHAPstories, CFstories
- 核心内容：把 SHAP / counterfactual explanations 转成 narrative explanations，让非技术用户更容易理解。
- 和 M2NA 的关系：可支撑“叙事解释比技术解释更易理解”的动机。
- 差异：
  - 它解释 AI decision。
  - M2NA 解释抽象概念机制。

### 3.4 TF1-EN-3M: Three Million Synthetic Moral Fables

- 年份/venue：arXiv 2025，v2 2026
- 链接：https://arxiv.org/abs/2504.20605
- 关键词：moral fables, scaffold, synthetic data, open LLM judges
- 核心内容：生成 300 万道德寓言，每个故事遵循 character -> trait -> setting -> conflict -> resolution -> moral 六槽 scaffold。
- 和 M2NA 的关系：可作为“寓言/短叙事 scaffold”参考。
- 差异：
  - 它是道德寓言生成。
  - M2NA 是抽象概念机制保真的隐性叙事类比生成。

## 4. 图到文本 / 结构到文本方向

### 4.1 Evaluating and Improving Graph to Text Generation with Large Language Models

- 年份/venue：NAACL 2025
- 链接：https://aclanthology.org/2025.naacl-long.513/
- 关键词：graph-to-text, LLM prompting, planning, grounding, PlanGTG
- 核心内容：系统评估开源 LLM 在 graph-to-text 上的能力，发现 LLM 在复杂图规划上困难；提出 PlanGTG，包括 reordering 和 attribution 子任务。
- 和 M2NA 的关系：非常重要。M2NA 输入是 mechanism graph，本质上借 graph-to-text，但输出不是 literal realization，而是 analogy realization。
- 可借鉴 baseline：
  - direct graph-to-text explanation
  - few-shot graph-to-text
  - planning/reordering + attribution

### 4.2 Unifying Structured Data as Graph for Data-to-Text Pre-Training

- 年份/venue：TACL 2024
- 链接：https://aclanthology.org/2024.tacl-1.12/
- 关键词：data-to-text, graph-to-text, structure-enhanced pre-training
- 核心内容：把表格、key-value、知识图谱统一成 graph format，并做 structure-enhanced D2T pretraining。
- 和 M2NA 的关系：提供结构到文本生成的技术背景，但 M2NA 还多了跨域类比和词汇隐藏。

### 4.3 Graph-Guided Textual Explanation Generation Framework

- 年份/venue：arXiv 2024 / ACL ARR 2025 submission
- 链接：https://arxiv.org/abs/2412.12318
- 关键词：graph-guided explanation, faithfulness, NLE
- 核心内容：用图指导自然语言解释生成，提高 faithfulness。
- 和 M2NA 的关系：可作为 faithfulness / graph-guided explanation 的相邻工作。

## 6. 推荐作为 baseline 的方法

### 必做 baseline

| Baseline | 说明 | 来源/理由 |
|---|---|---|
| Direct Explanation | `G_c -> literal explanation` | graph-to-text / concept explanation |
| Direct Analogy | `concept + definition -> explicit analogy` | analogy generation / educational analogy |
| Vanilla Narrative | `concept + definition -> short story` | 普通 LLM 讲故事 |
| Graph-to-Text | `G_c -> text`，不做隐性类比 | NAACL 2025 Graph-to-Text |
| Analogical Prompting | 先自生成类似案例，再生成 | ICLR 2024 |
| Plan-then-Write | 先生成 narrative plan，再写 | narrative generation common baseline |

### 消融 baseline

| Variant | 检验点 |
|---|---|
| w/o mechanism graph | 机制图是否必要 |
| w/o forbidden lexical set | 词汇隐藏约束是否有效 |
| w/o alignment planning | 结构映射规划是否有效 |
| w/o self-revision | 自检是否降低泄露和遗漏 |
| w/o source-domain selection | 源域选择是否影响映射质量 |

## 8. 当前 gap 句子

可以直接放进论文：

```text
Recent work has studied analogy generation, narrative analogy recognition, educational analogies for concept learning, and graph-to-text generation. However, these lines of work do not address whether language models can transform an abstract concept mechanism into a lexically concealed narrative analogy with an explicit structure-level alignment. M2NA fills this gap by requiring models to preserve conceptual mechanisms, avoid high-leakage terminology, and realize the mechanism through an alignable short narrative.
```

中文：

> 近期工作已经研究了类比生成、叙事类比识别、面向概念学习的教育类比，以及图到文本生成。但这些工作尚未研究语言模型能否将抽象概念机制转化为一种词汇隐藏的叙事类比，并同时给出结构级映射。M2NA 填补了这一空白：它要求模型在保留概念机制的同时避免高泄露术语，并通过可对齐的短叙事实现该机制。
